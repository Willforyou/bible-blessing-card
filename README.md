# 🧧 新春蒙福 - 聖經金句祝福卡 (Bible Blessing Card)

這是一個專為農曆新年設計的線上祝福應用程式，結合了精美的視覺特效與聖經寶貴的應許，讓您在數位時代也能傳遞最溫暖的福音祝福。

## ✨ 特色功能

- **喜慶視覺設計**：充滿年節氣氛的深紅金主題，搭配動態升空的燈籠與星光特效。
- **3D 互動體驗**：擬真的 3D 翻牌動畫，增加領取祝福的儀式感。
- **震撼慶祝特效**：翻開卡片時觸發華麗的絲帶噴發與金箔碎屑效果。
- **精選聖經金句**：每一則經文都配有專屬小圖示與深度寓意解析。
- **響應式排版**：完美支援手機與電腦瀏覽，隨時隨地分享祝福。

## 🛠️ 技術棧

- **Vite + React**：極速的開發與運行體驗。
- **Canvas-confetti**：打造絲帶噴發動效。
- **Vanilla CSS**：純手寫 3D 變換與粒子動畫，極致流暢。

## 🚀 快速上手

1. **複製專案**
   ```bash
   git clone https://github.com/你的帳號/bible-blessing-card.git
   ```
2. **安裝依賴**
   ```bash
   npm install
   ```
3. **啟動開發服務**
   ```bash
   npm run dev
   ```
